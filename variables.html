Variables

Introducción a las Variables

Las variables son contenedores para almacenar datos en un programa. Pueden contener datos que pueden ser constantes o variables, como números, caracteres, sonidos, imágenes, entre otros.

Cómo Declarar una Variable

Para crear una variable en Microbit, sigue estos pasos:

  1. Dirígete al Bloque de variables.
  2. Crea una nueva variable y asigna un nombre.
  3. Utiliza la variable en tu programa.

Una variable puede tener tres bloques:

  • El primer bloque se utiliza como entrada para asignar el valor de la variable.
  • El segundo bloque establece el valor inicial de la variable.
  • El tercer bloque cambia el valor de la variable sumando el valor de la entrada.

Alcance de las Variables

Las variables pueden ser globales o locales. Una variable es global a menos que esté declarada dentro de una definición de función. Las variables globales son visibles y están disponibles para todas las instrucciones de un código.

Las variables locales solo son visibles y están disponibles dentro de la función en la que están definidas.
